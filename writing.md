# Writing Skill (OpenClaw Style)

> 콘텐츠 작성 워크플로우. "글쓰기", "블로그", "문서 작성", "콘텐츠", "기사" 트리거.

## 콘텐츠 유형별 가이드

### 기술 블로그
```markdown
# [주제]: [핵심 가치/결과]

## TL;DR
- 핵심 포인트 1
- 핵심 포인트 2
- 핵심 포인트 3

## 배경/문제
왜 이 글을 쓰게 되었는지

## 해결 과정
1. 시도 1
2. 시도 2
3. 최종 해결책

## 코드/예제
```코드 블록```

## 결론
- 배운 점
- 다음 단계
```

## 글쓰기 원칙

### 1. 독자 중심
- 누가 읽을 것인가?
- 무엇을 알고 싶어하는가?
- 어떤 수준인가?

### 2. 구조화
- 서론: 왜 읽어야 하는지
- 본론: 핵심 내용
- 결론: 요약 + 다음 액션

### 3. 명확성
- 한 문장 = 한 아이디어
- 전문 용어는 설명 추가
- 예제로 이해도 높이기

## SEO 최적화

### 제목
- 키워드 포함
- 60자 이내
- 숫자/질문 활용

### 메타 설명
- 155자 이내
- 행동 유도 포함

### 구조
- H1 하나만
- H2, H3 계층적 사용
- 이미지 alt 태그

## 체크리스트

- [ ] 핵심이 전달되는가
- [ ] 오탈자 검사
- [ ] 링크 동작 확인
