## UX 패턴 정리

### 학습 플로우 (최적 패턴)
```
1. 대시보드 (진도 + 오늘 할 일)
   ↓
2. 문제 풀기 (10문제 세트)
   ↓
3. 즉각 피드백 (정답/오답 + 풀이)
   ↓
4. 결과 화면 (XP + 뱃지 + 코인)
   ↓
5. 보상 연출 (레벨업/뱃지 애니메이션)
   ↓
6. 대시보드 복귀 (진도 업데이트)
```

### 리텐션 전략
```yaml
Day 0: 온보딩 (레벨 테스트, 목표 설정)
Day 1: 푸시 알림 ("어제 시작한 학습을 이어가세요!")
Day 3: 스트릭 보상 (3일 연속 뱃지)
Day 7: 주간 리포트 (학부모에게도)
Day 14: 리그 시작 (경쟁 요소)
Day 30: 월간 보상 (특별 뱃지)
```

### 교사 대시보드 필수 지표
```yaml
반 전체:
  - 평균 진도율
  - 평균 정답률
  - 오늘 학습 현황 (완료/미완료)
  - 취약 단원 (반 전체)

학생 개인:
  - 학습 진도 (단원별)
  - 정답률 추이 (주간/월간)
  - 취약 유형 분석
  - 학습 시간
  - 스트릭
  - 오답 노트
```

### 학부모 리포트 필수 항목
```yaml
일간:
  - 오늘 풀이 수 / 정답률
  - 학습 시간
  - 스트릭 현황

주간:
  - 주간 정답률 추이
  - 가장 잘한 영역 / 취약 영역
  - 반 내 위치 (상위 %)
  - 추천 보충 학습

월간:
  - 전체 진도율
  - 성장 추이 그래프
  - 성취 뱃지 목록
  - 교사 코멘트
```

## 가격 정책 벤치마크

```yaml
무료 (Freemium):
  - 하루 10문제 제한
  - 광고 표시
  - 기본 게이미피케이션
  - 기본 리포트

프리미엄 개인:
  - 가격: ₩9,900/월 (연 ₩89,000)
  - 무제한 문제
  - 광고 제거
  - AI 힌트
  - 상세 리포트
  - 오답 노트

프리미엄 가족:
  - 가격: ₩14,900/월
  - 자녀 3명까지
  - 학부모 대시보드
  - 형제자매 비교 리포트

학교/기관:
  - 가격: 학생당 ₩5,000/월
  - 최소 30명
  - 교사 대시보드
  - 반 관리
  - 커리큘럼 연동
  - 관리자 계정
```
