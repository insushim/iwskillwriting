# 한국 교육과정 참조 (2022 개정 교육과정)

> 교육부 고시 제2022-33호 기반, NCIC(ncic.re.kr) 참조

## 초등학교 수학 — 영역별 핵심 내용

### 수와 연산

#### 1학년
```yaml
1학기:
  - 9까지의 수
  - 50까지의 수
  - 덧셈과 뺄셈 (합이 9 이하)
2학기:
  - 100까지의 수
  - 덧셈과 뺄셈 (받아올림/받아내림 없음)
```

#### 2학년
```yaml
1학기:
  - 세 자리 수
  - 덧셈과 뺄셈 (두 자리 + 두 자리)
  - 곱셈 (곱셈구구 2~5)
2학기:
  - 네 자리 수
  - 곱셈구구 (6~9, 1, 0)
  - 덧셈과 뺄셈 (세 자리)
```

#### 3학년
```yaml
1학기:
  - 덧셈과 뺄셈 (세 자리 + 세 자리)
  - 곱셈 (두 자리 × 한 자리)
  - 나눗셈 (기본)
  - 분수 (단위분수)
2학기:
  - 곱셈 (세 자리 × 한 자리)
  - 나눗셈 (나머지 있는)
  - 분수 (진분수, 가분수, 대분수)
  - 소수 (소수 한 자리)
```

#### 4학년
```yaml
1학기:
  - 큰 수 (만, 억, 조)
  - 곱셈과 나눗셈 (세 자리 × 두 자리)
  - 분수의 덧셈과 뺄셈 (동분모)
  - 소수의 덧셈과 뺄셈
2학기:
  - 분수의 덧셈과 뺄셈 (이분모)
  - 소수의 곱셈
  - 어림하기 (반올림, 올림, 버림)
```

#### 5학년
```yaml
1학기:
  - 자연수의 혼합 계산
  - 약수와 배수
  - 약분과 통분
  - 분수의 덧셈과 뺄셈 (이분모)
  - 분수의 곱셈
2학기:
  - 분수의 나눗셈
  - 소수의 곱셈
  - 소수의 나눗셈
  - 평균과 가능성
```

#### 6학년
```yaml
1학기:
  - 분수의 나눗셈
  - 소수의 나눗셈
  - 비와 비율
  - 비례식과 비례배분
2학기:
  - 분수와 소수의 혼합 계산
  - 비례식 활용
  - 정비례와 반비례
```

### 도형

#### 학년별 핵심 도형 개념
```yaml
1학년: 여러 가지 모양 (□, △, ○)
2학년: 원, 삼각형, 사각형, 오각형, 육각형
3학년: 원의 구성 요소, 각, 직각, 직각삼각형
4학년: 삼각형 분류, 사각형 분류, 다각형, 정다각형
5학년: 합동과 대칭, 직육면체, 정육면체
6학년: 각기둥, 각뿔, 원기둥, 원뿔, 구
```

### 측정

```yaml
1학년: 비교하기 (길이, 높이, 넓이, 들이)
2학년: cm, m, 시각과 시간
3학년: mm, km, 들이(L, mL), 무게(kg, g), 초
4학년: 각도, 넓이(cm², m²), 어림하기
5학년: 둘레와 넓이 (삼각형, 사각형), 부피(cm³)
6학년: 원의 넓이, 원기둥의 겉넓이와 부피
```

### 규칙성

```yaml
1학년: 규칙 찾기
2학년: 규칙적인 무늬, 수 배열의 규칙
3학년: 규칙과 대응
4학년: 규칙 찾기 (수의 배열)
5학년: 규칙과 대응 (두 양 사이 관계)
6학년: 비례식, 정비례/반비례
```

### 자료와 가능성

```yaml
1학년: 분류하기
2학년: 표와 그래프
3학년: 그림그래프
4학년: 막대그래프, 꺾은선그래프
5학년: 평균, 가능성
6학년: 비율그래프 (띠그래프, 원그래프)
```

## 초등학교 국어

### 영역
```yaml
듣기·말하기:
  1-2학년: 바른 자세, 인사말, 대화
  3-4학년: 요약하며 듣기, 회의, 발표
  5-6학년: 토의, 토론, 면담

읽기:
  1-2학년: 글자, 낱말, 문장 읽기
  3-4학년: 사실과 의견, 요약
  5-6학년: 추론, 비판적 읽기

쓰기:
  1-2학년: 글자 쓰기, 문장 쓰기
  3-4학년: 중심 문장, 문단 쓰기
  5-6학년: 논설문, 보고서

문법:
  1-2학년: 자모, 낱말
  3-4학년: 문장 성분, 높임말
  5-6학년: 관용어, 어휘

문학:
  1-2학년: 동시, 동화 감상
  3-4학년: 이야기 구성, 인물
  5-6학년: 갈래, 비유, 상징
```

## 초등학교 영어 (3~6학년)

### 학년별 핵심 내용
```yaml
3학년:
  - 알파벳 대소문자
  - 기본 인사, 소개
  - 색, 숫자, 사물 이름
  - 간단한 문장 (I am, This is)

4학년:
  - 요일, 날씨, 감정
  - 좋아하는 것 표현 (I like)
  - 위치 표현 (in, on, under)
  - 간단한 질문과 대답

5학년:
  - 시간, 날짜 표현
  - 할 수 있는 것 (I can)
  - 일과 표현
  - 짧은 글 읽기/쓰기

6학년:
  - 비교 표현
  - 과거 경험 (I went to)
  - 미래 계획 (I will)
  - 짧은 글 쓰기
```

## 초등학교 과학

### 학년별 핵심 단원
```yaml
3학년:
  1학기: 물질의 성질, 동물의 한살이, 자석의 이용, 지구의 모습
  2학기: 동물의 생활, 지표의 변화, 물질의 상태, 소리의 성질

4학년:
  1학기: 식물의 한살이, 물의 상태 변화, 그림자와 거울, 화산과 지진
  2학기: 식물의 생활, 물의 여행, 거울과 그림자, 지구와 달

5학년:
  1학기: 온도와 열, 태양계와 별, 용해와 용액, 다양한 생물
  2학기: 생물과 환경, 날씨와 우리 생활, 물체의 운동, 산과 염기

6학년:
  1학기: 지구와 달의 운동, 여러 가지 기체, 식물의 구조와 기능, 빛과 렌즈
  2학기: 전기의 이용, 계절의 변화, 연소와 소화, 우리 몸의 구조와 기능
```

## 성취기준 코드 체계

```
[학년군]-[영역]-[순번]
예: 2수01-01 = 1~2학년군, 수와 연산, 1번째 성취기준

영역 코드:
  수: 수와 연산
  도: 도형
  측: 측정
  규: 규칙성
  자: 자료와 가능성
```
