# 소설/웹소설 작성 지원

## 장르별 템플릿

### 판타지
```markdown
# [제목]

## 세계관 설정
- 마법 체계:
- 종족:
- 국가/세력:
- 역사적 배경:

## 주인공
- 이름:
- 나이:
- 능력/스킬:
- 성격:
- 목표:
- 약점:

## 플롯 구조
1. 발단: 평범한 일상 / 소환 / 각성
2. 전개: 능력 획득, 동료 만남
3. 위기: 강적 등장, 배신
4. 절정: 최종 결전
5. 결말: 새로운 시작
```

### 로맨스
```markdown
# [제목]

## 설정
- 배경: 현대 / 사극 / 판타지
- 분위기: 달달 / 짠내 / 코믹

## 주인공 (여주/남주)
- 여주:
  - 이름, 나이, 직업
  - 성격, 트라우마
- 남주:
  - 이름, 나이, 직업
  - 성격, 비밀

## 관계 발전 단계
1. 첫 만남 (인상적인 장면)
2. 재회 / 엮임
3. 밀당 / 오해
4. 감정 인식
5. 고백 / 위기
6. 해피엔딩
```

### 회귀/빙의물
```markdown
# [제목]

## 회귀/빙의 설정
- 회귀 시점:
- 원래 결말:
- 바꾸고 싶은 것:
- 가진 정보/능력:

## 버터플라이 이펙트
- 1차 개입 → 결과
- 2차 개입 → 결과
- 예상치 못한 변화:

## 긴장감 요소
- 원작 지식의 한계
- 새로운 변수
- 시간 제한
```

## 글쓰기 팁

### 첫 문장 훅
```
❌ "오늘도 평범한 하루였다."
✅ "내 손에서 피가 뚝뚝 떨어지고 있었다."
✅ "죽었다고 생각했는데, 눈을 떠보니 10년 전이었다."
```

### 대화문 작성
```
❌ "안녕하세요." 그녀가 말했다.
   "네, 안녕하세요." 그가 대답했다.

✅ "안녕하세요."
   그녀의 목소리가 떨리고 있었다.
   "...늦었군요."
   그는 시계를 확인하며 무심하게 내뱉었다.
```

### Show, Don't Tell
```
❌ 그는 화가 났다.
✅ 그의 주먹이 부르르 떨렸다. 이를 악문 턱 근육이 불끈 솟았다.

❌ 그녀는 슬펐다.
✅ 그녀는 고개를 숙인 채 한참을 움직이지 않았다.
   바닥에 물방울이 하나, 둘 떨어졌다.
```
