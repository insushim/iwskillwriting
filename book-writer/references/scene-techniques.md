## Scene Construction (장면 구성)

### Dwight Swain의 Scene-Sequel 상세 가이드

**Scene (장면) — 행동 단위:**

```
목표 (Goal) → 갈등 (Conflict) → 재앙 (Disaster)
```

| 요소 | 설명 | 질문 |
|------|------|------|
| Goal | POV 캐릭터가 이 장면에서 원하는 것 | "무엇을 성취하려 하는가?" |
| Conflict | 목표를 방해하는 장애물 | "무엇이 막고 있는가?" |
| Disaster | 상황을 악화시키는 결과 | "어떻게 더 나빠지는가?" |

**Sequel (후속) — 반응 단위:**

```
반응 (Reaction) → 딜레마 (Dilemma) → 결정 (Decision)
```

| 요소 | 설명 | 질문 |
|------|------|------|
| Reaction | 재앙에 대한 감정적 반응 | "무엇을 느끼는가?" |
| Dilemma | 선택지들 사이의 고민 | "어떤 선택이 가능한가?" |
| Decision | 다음 행동의 결정 | "무엇을 하기로 하는가?" |

### 장면 유형별 가이드

| 유형 | 목적 | 비율 | 길이 |
|------|------|------|------|
| **액션 장면** | 외적 갈등, 긴장 | 30-40% | 짧게, 빠르게 |
| **반응 장면** | 감정 처리, 성찰 | 15-20% | 중간 |
| **대화 장면** | 정보 교환, 관계 | 25-35% | 유연하게 |
| **묘사 장면** | 분위기, 세계관 | 5-10% | 짧게 |
| **전환 장면** | 시간/장소 이동 | 5-10% | 최소한으로 |
| **계시 장면** | 비밀/반전 공개 | 5-10% | 임팩트 있게 |

## Try-Fail Cycles 상세

### 3회 시도-실패 패턴 (가장 일반적)

```
시도 1: 가장 확실한 방법 → 실패 (예상치 못한 이유로)
시도 2: 새로운 접근 → 거의 성공, 하지만 더 큰 문제 발생
시도 3: 최후의 수단 + 내적 성장 → 성공 (대가를 치르며)
```

### Yes-But / No-And 전개 패턴

| 패턴 | 의미 | 효과 | 예시 |
|------|------|------|------|
| **Yes, but...** | 성공하지만 새 문제 | 긴장 유지 | 탈출했지만 동료가 잡혔다 |
| **No, and...** | 실패하고 상황 악화 | 위기 고조 | 문이 잠겼고 물이 차오른다 |
| **Yes, and...** | 성공하고 추가 이득 | 보상 (드물게) | 보물을 찾았고 지도도 있다 |
| **No, but...** | 실패하지만 실마리 | 희망 유지 | 적을 못 잡았지만 단서를 얻었다 |

## 대화문 마스터 가이드

### David Mamet의 대화 원칙
1. 모든 대화에는 **목적**이 있어야 한다 (정보 전달만은 불충분)
2. 캐릭터는 대화로 **무언가를 얻으려** 해야 한다
3. **서브텍스트**가 없는 대화는 제거하라
4. "실제로 사람들이 말하는 방식"이 아니라 **"실제처럼 들리는 방식"**으로 쓰라

### 서브텍스트 기법

| 기법 | 설명 | 예시 |
|------|------|------|
| 간접 답변 | 질문에 직접 답하지 않기 | "사랑해?" → "배고프지 않아?" |
| 화제 전환 | 불편한 주제 회피 | "아버지는?" → "날씨가 좋네" |
| 행동 모순 | 말과 행동이 다르기 | "괜찮아" (눈물을 닦으며) |
| 과잉 반응 | 사소한 것에 크게 반응 | "컵을 깼어" → "항상 이런 식이야!" |
| 침묵 | 대답하지 않음 자체가 메시지 | "…" (긴 침묵) |
| 반복 | 같은 말을 반복해서 의미 변화 | 처음 "사랑해"와 마지막 "사랑해"의 무게 |

### 대화 태그 원칙

**기본 규칙:**
- "말했다(said)"를 기본으로 사용 (독자가 투명하게 넘어감)
- 화려한 태그는 1페이지에 1회 이하
- 행동 비트로 태그 대체 가능

**좋은 예:**
```
"가지 마." 그녀가 소맷자락을 잡았다.
"왜?"
"이유 같은 건 없어." 그녀의 손가락이 하얗게 변해 있었다.
```

**나쁜 예:**
```
"가지 마!" 그녀가 애절하게 외쳤다.
"왜?" 그가 차갑게 물었다.
"이유 같은 건 없어." 그녀가 슬프게 속삭였다.
```

## 묘사(Description) 전략

### 감각 묘사 밀도 가이드

**장면 시작 시: 2-3개 감각**
```
[시각] 해질녘의 주황빛이 유리창에 반사되고 있었다.
[청각] 멀리서 기차의 경적이 울렸다.
[후각] 커피 찌꺼기의 쓴 냄새가 공기에 배어 있었다.
```

**액션 중: 1-2개 감각 (속도 유지)**
```
[촉각] 칼날이 팔뚝을 스쳤다. 뜨거웠다.
```

**감정 장면: 내적 감각 + 1개 외적 감각**
```
[내적] 목구멍이 조여왔다.
[시각] 그의 뒷모습이 점점 작아졌다.
```

### Show vs Tell 변환 가이드

| Tell (금지) | Show (권장) |
|------------|-----------|
| 그녀는 화가 났다 | 그녀의 손가락이 테이블을 두드렸다. 리듬이 점점 빨라졌다 |
| 그는 슬펐다 | 그는 창밖을 보았다. 빗방울이 유리를 타고 흘러내렸다 |
| 그녀는 긴장했다 | 그녀는 반지를 돌렸다. 한 바퀴, 두 바퀴, 세 바퀴 |
| 분위기가 어색했다 | 누군가 목을 가다듬었다. 시계 소리만 교실을 채웠다 |
| 그는 용감했다 | 그의 다리가 떨렸지만, 한 발 더 앞으로 내딛었다 |

### 날씨/환경 묘사와 감정 대응

| 감정 | 환경 | 효과 |
|------|------|------|
| 고독 | 안개, 빈 들판 | 고전적 대응 (주의: 클리셰) |
| 갈등 | 폭풍 전 고요 | 긴장 고조 |
| 희망 | 비 온 뒤 맑음 | 전환 암시 |
| **역대응 (고급)** | 감정과 반대 환경 | 아이러니 효과 |

**역대응 예시:** 장례식 날의 화창한 날씨, 사랑 고백 순간의 폭우

## 오프닝 & 클로징 기법

### 강력한 오프닝 7가지 유형

| 유형 | 설명 | 예시 패턴 |
|------|------|----------|
| **In Medias Res** | 액션 한복판 | "칼날이 목을 스치는 순간—" |
| **강렬한 이미지** | 시각적 충격 | "하늘이 두 쪽으로 갈라졌다." |
| **미스터리** | 풀리지 않는 질문 | "그녀가 사라진 것은 화요일이었다." |
| **대화** | 즉시 캐릭터 진입 | "'다시는 돌아오지 마.' 아버지가 말했다." |
| **독백/선언** | 강한 목소리 | "나는 세 번 죽었다." |
| **세계관 제시** | 규칙이 다른 세계 | "마법이 사라진 지 백 년이 지났다." |
| **아이러니** | 기대 반전 | "그날은 평범한 월요일이었다. 적어도 시작은." |

### 효과적인 클리프행어 유형

| 유형 | 메커니즘 | 웹소설 효과 |
|------|---------|-----------|
| **위기 예고** | 다음 장면의 위험 암시 | 매우 높음 |
| **비밀 공개** | 충격적 정보 마지막 줄 | 매우 높음 |
| **선택의 갈림길** | 결정하지 않은 채 끝 | 높음 |
| **타이머** | 시간 제한 설정 | 높음 |
| **감정 반전** | 예상과 반대 감정 | 중간 |
| **새 캐릭터** | 의문의 인물 등장 | 중간 |

## 페이싱(Pacing) 제어 도구

### 빠르게 만드는 기법
- 짧은 문장, 짧은 문단
- 대화 중심
- 액션 동사
- 한 문장 문단
- 챕터 짧게 끊기

### 느리게 만드는 기법
- 감각 묘사 확장
- 내적 독백 삽입
- 배경 설명
- 긴 문장, 복합 구조
- 풍경/분위기 묘사

### 장르별 적정 페이스

| 장르 | 평균 장면 길이 | 대화:묘사 비율 | 클리프행어 빈도 |
|------|-------------|-------------|-------------|
| 스릴러 | 1,000-2,000자 | 60:40 | 매 장면 |
| 로맨스 | 2,000-3,000자 | 50:50 | 매 챕터 |
| 판타지 | 2,500-4,000자 | 40:60 | 주요 장면 |
| 문학소설 | 3,000-5,000자 | 35:65 | 드물게 |
| 웹소설 | 3,000-5,000자 | 55:45 | 매 화 |

## John Truby의 22단계 구조 (The Anatomy of Story)

1. 자기 계시/욕구 (Self-revelation, need)
2. 유령과 이야기 세계 (Ghost and story world)
3. 약점과 욕구 (Weakness and need)
4. 욕망 (Desire)
5. 적대자 (Opponent)
6. 계획 (Plan)
7. 동맹 (Ally)
8. 강제된 동맹 (Forced ally)
9. 1차 계시와 결정 (First revelation and decision)
10. 계획 변경 (Changed plan)
11. 드라이브 (Drive)
12. 공격과 반격 (Attack by ally)
13. 겉보기 패배 (Apparent defeat)
14. 2차 계시와 결정 (Second revelation and decision)
15. 관객 계시 (Audience revelation)
16. 3차 계시와 결정 (Third revelation and decision)
17. 문 (Gate), 벽 (Wall), 방문 (Visit)
18. 전투 (Battle)
19. 자기 계시 (Self-revelation)
20. 도덕적 결정 (Moral decision)
21. 새로운 균형 (New equilibrium)
22. 최종 이미지 (Final image)

## Lisa Cron의 뇌과학 기반 스토리 원칙 (Wired for Story)

### 7가지 핵심 원칙

1. **인과관계**: 모든 장면은 "때문에(because)" 연결되어야 한다 ("그리고(and then)" 아님)
2. **내면 변화**: 외적 사건보다 캐릭터의 내적 반응이 스토리
3. **기대와 보상**: 독자의 예측을 만족시키거나 더 좋은 것으로 대체
4. **구체성**: 추상적 개념은 구체적 디테일로 전환
5. **갈등의 필연성**: 갈등은 캐릭터의 결점에서 자연발생
6. **감정적 투자**: 독자가 결과를 걱정해야 페이지를 넘긴다
7. **변화의 불가역성**: 모든 장면은 캐릭터를 돌아올 수 없는 곳으로

## Donald Maass의 긴장도 높이기 기법 (Writing the Breakout Novel)

### 모든 페이지에 긴장을 넣는 5가지 방법

1. **미시적 긴장**: 모든 대화 라인에 갈등/불일치 삽입
2. **하위 목표**: 큰 목표 외에 장면별 작은 갈등
3. **시간 압박**: 기한/데드라인 설정
4. **정보 비대칭**: 독자/캐릭터가 서로 다른 것을 알게
5. **내적 모순**: 캐릭터가 원하는 것과 필요한 것의 충돌
