# Worldbuilding Guide Reference

세계관 구축을 위한 종합 가이드. 마법 체계, 경제, 정치, 종교, 기술, 종족 설계.

---

## 1. Brandon Sanderson의 마법 3법칙

### 제1법칙: 해결 능력 = 독자 이해도

> 작가가 마법으로 갈등을 해결하는 능력은 독자가 그 마법을 이해하는 정도에 비례한다.

| 구분 | Hard Magic | Soft Magic |
|------|-----------|-----------|
| **규칙** | 명확하고 일관된 규칙 | 모호하고 신비로운 |
| **독자 이해** | 높음 | 낮음 |
| **갈등 해결** | 마법으로 해결 가능 | 마법은 문제를 만듦 |
| **효과** | 지적 만족감 | 경외감, 신비감 |
| **예시** | 미스트본의 합금술 | 반지의 제왕의 간달프 |

**체크리스트:**
- [ ] 마법으로 갈등을 해결하는 장면에서, 그 마법의 규칙이 사전에 설명되었는가?
- [ ] 독자가 "반칙"이라고 느낄 해결이 없는가?
- [ ] 마법의 한계가 명확하여 긴장감이 유지되는가?

### 제2법칙: 한계 > 능력

> 마법의 약점과 한계가 능력 자체보다 더 흥미롭다.

**비용/대가 유형:**
| 비용 유형 | 설명 | 예시 |
|---------|------|------|
| 물리적 | 체력, 건강 소모 | 피로, 수명 단축 |
| 정신적 | 정신력, 기억 소모 | 망각, 광기 |
| 자원 | 희귀 재료 필요 | 마석, 특수 금속 |
| 도덕적 | 윤리적 대가 | 생명 희생, 금기 위반 |
| 상황적 | 특정 조건 필요 | 보름달, 특정 장소 |
| 시간적 | 준비/회복 시간 | 긴 주문, 쿨다운 |

**체크리스트:**
- [ ] 각 능력에 명확한 비용/한계가 있는가?
- [ ] 한계가 갈등을 생성하는가?
- [ ] 능력이 만능이 아닌가?

### 제3법칙: 확장 전 심화

> 새로운 마법 요소를 추가하기 전에, 기존 요소를 창의적으로 활용하라.

**체크리스트:**
- [ ] 기존 능력의 비일반적 활용을 탐구했는가?
- [ ] 새 능력이 기존 체계와 논리적으로 연결되는가?
- [ ] 체계가 과도하게 복잡해지지 않았는가?

## 2. 경제 월드빌딩

### 경제 체계 설계 체크리스트

**화폐 시스템:**
- [ ] 화폐의 종류와 가치 체계 (금화/은화/동화 등)
- [ ] 화폐의 물리적 형태 (주화, 지폐, 보석, 디지털)
- [ ] 환율 체계 (다국가 설정 시)
- [ ] 물가 기준표 (빵 1개 = ?, 검 1자루 = ?, 집 1채 = ?)

**경제 구조:**
| 요소 | 질문 | 설정 예시 |
|------|------|---------|
| 생산 | 주요 산업은? | 농업, 광업, 마법 제조 |
| 무역 | 주요 무역 루트는? | 해상로, 대상단, 포탈 |
| 자원 | 희귀 자원은? | 마석, 특수 금속, 약초 |
| 계급 | 경제적 계층은? | 귀족/상인/농민/노예 |
| 세금 | 세금 체계는? | 생산 10%, 마법 사용세 |
| 통화 | 통화 안정성은? | 인플레이션, 위조 문제 |

## 3. 정치 월드빌딩

### 통치 체계 유형

| 체계 | 권력 원천 | 갈등 유발 | 적합 장르 |
|------|---------|---------|----------|
| 봉건제 | 토지/혈통 | 계승 분쟁, 영지 전쟁 | 중세 판타지 |
| 절대왕정 | 신권/무력 | 폭정, 혁명 | 궁정 음모 |
| 공화제 | 선거/원로원 | 파벌, 부패, 선거 | 정치 스릴러 |
| 신정정치 | 종교/신 | 이단 박해, 교리 갈등 | 다크 판타지 |
| 마법정치 | 마법 능력 | 비마법사 차별, 반란 | 하이 판타지 |
| 군사정권 | 군사력 | 쿠데타, 저항운동 | 디스토피아 |
| 길드연합 | 경제력/기술 | 독점, 경쟁, 암살 | 도시 판타지 |
| 부족연합 | 전통/혈연 | 부족간 갈등, 통일 | 에픽 판타지 |
| 디스토피아 | 감시/기술 | 저항, 탈출 | SF, 디스토피아 |

### 정치 갈등 설계

| 갈등 유형 | 설명 | 예시 |
|---------|------|------|
| 계승 | 왕좌/지도자 자리 다툼 | 왕자들의 전쟁 |
| 영토 | 국경/자원 분쟁 | 두 왕국의 전쟁 |
| 이념 | 가치관/세계관 충돌 | 마법 자유 vs 통제 |
| 계급 | 지배층 vs 피지배층 | 평민 반란 |
| 종족 | 종족 간 긴장 | 엘프-드워프 갈등 |
| 외교 | 동맹/배신 | 조약 파기 |

## 4. 종교/신앙 월드빌딩

### 신앙 체계 설계

| 요소 | 설명 | 설정 질문 |
|------|------|---------|
| 신의 존재 | 실제? 추상? | 신이 물리적으로 개입하는가? |
| 다신교/일신교 | 신의 수 | 판테온? 유일신? 이원론? |
| 성직자 | 종교 기관 | 교단 구조? 권력? 재산? |
| 의식 | 종교 행사 | 기도, 제사, 축제, 금기 |
| 교리 | 믿음 체계 | 사후세계? 윤회? 구원? |
| 이단 | 대립 종교 | 분파, 이단, 무신론 |
| 마법과 관계 | 신성 마법? | 축복? 저주? 신의 힘? |

### 창세 신화 템플릿

```
태초에 [창조자/힘]이 [세계의 원형]을 만들었다.
[창조 방식: 말씀/노동/전투/꿈]으로 [세계의 핵심 요소]가 생겨났다.
[최초의 존재들]이 [역할/임무]을 부여받았다.
[타락/재앙의 원인]으로 인해 [세계의 결함/갈등]이 시작되었다.
[예언/약속]에 따르면, [미래의 사건]이 일어날 것이다.
```

## 5. 문화 월드빌딩

### 문화 설계 체크리스트

| 범주 | 항목 |
|------|------|
| **언어** | 인사말, 욕설, 존칭, 속어, 문자 체계 |
| **의식주** | 주식, 의복 재질/스타일, 건축 양식 |
| **가족** | 핵가족/대가족, 결혼 관습, 상속 규칙 |
| **교육** | 문해율, 교육 기관, 도제 시스템 |
| **예술** | 음악, 미술, 문학, 공연 전통 |
| **금기** | 절대 해서는 안 되는 것, 불경한 것 |
| **축제** | 계절 축제, 종교 행사, 통과의례 |
| **장례** | 매장/화장/수장, 애도 기간, 사후 관념 |
| **법률** | 범죄 정의, 재판 방식, 처벌 체계 |
| **젠더** | 성 역할, 동성 관계, 성별 권력 구조 |

## 6. 기술/마법 수준 설정

### 기술 수준표

| 레벨 | 시대 대응 | 특징 | 적합 배경 |
|------|---------|------|---------|
| 1 | 석기시대 | 불, 돌 도구 | 원시 판타지 |
| 2 | 청동기 | 금속 무기, 농경 | 고대 문명 |
| 3 | 중세 초기 | 제철, 봉건 | 로우 판타지 |
| 4 | 중세 후기 | 석궁, 판금갑 | 하이 판타지 |
| 5 | 르네상스 | 화약, 인쇄 | 마법+기술 혼합 |
| 6 | 산업혁명 | 증기기관, 공장 | 스팀펑크 |
| 7 | 근대 | 전기, 자동차 | 어반 판타지 |
| 8 | 현대 | 컴퓨터, 인터넷 | 현대 판타지/SF |
| 9 | 근미래 | AI, 우주여행 | SF |
| 10 | 원미래 | FTL, 초광속 | 스페이스 오페라 |

### 마법과 기술의 관계

| 관계 유형 | 설명 | 예시 |
|---------|------|------|
| 대체 | 마법이 기술을 대체 | 전기 대신 마법 조명 |
| 공존 | 마법과 기술이 공존 | 마법 엔진 열차 |
| 대립 | 마법과 기술이 충돌 | 마법사 vs 과학자 |
| 혼합 | 마법이 기술에 통합 | 마법 프로그래밍 |
| 비밀 | 마법이 기술 뒤에 숨음 | 현대 도시의 숨겨진 마법 |

## 7. 종족 설계

### 종족 설계 템플릿

```
종족명:
기원:
평균 수명:
신체적 특징:
  - 키/체형:
  - 특수 능력:
  - 약점:
문화적 특징:
  - 가치관:
  - 사회 구조:
  - 타 종족과의 관계:
마법 친화도:
기술 수준:
```

### 종족 간 관계 매트릭스 예시

| | 인간 | 엘프 | 드워프 | 오크 |
|---|------|------|--------|------|
| 인간 | - | 경외/질투 | 교역 | 적대 |
| 엘프 | 멸시/보호 | - | 냉대 | 적대 |
| 드워프 | 교역/존중 | 냉대 | - | 적대 |
| 오크 | 약탈 | 증오 | 전쟁 | - |

## 8. 지리 설계

### 지리 체크리스트

| 요소 | 설정 항목 |
|------|---------|
| **대륙/도시** | 이름, 위치, 크기, 인구 |
| **기후** | 온대/열대/극지, 계절 패턴 |
| **지형** | 산맥, 강, 사막, 숲, 바다 |
| **자원** | 광물, 식물, 마법 자원 분포 |
| **경로** | 주요 도로, 항로, 위험 지대 |
| **국경** | 자연적 경계 (강, 산맥), 정치적 경계 |

### 이동 시간 기준표

| 이동 수단 | 하루 이동 거리 | 비고 |
|---------|-------------|------|
| 도보 | 25-35km | 평지, 도로 |
| 기마 | 50-80km | 교체 말 없이 |
| 마차 | 30-50km | 도로 상태 의존 |
| 범선 | 100-200km | 풍향 의존 |
| 갤리선 | 60-100km | 노 + 돛 |
| 비행 (판타지) | 200-500km | 생물/마법 의존 |
| 전이문/포탈 | 즉시 | 제약 조건 설정 필요 |
