## 5. 에세이 구조 패턴

### 기본형 (5단계)
```
1. 구체적 장면/경험으로 시작 (감각 묘사)
2. 성찰과 확장 (경험 → 의미)
3. 지식/인용으로 깊이 더하기 (선택)
4. 보편적 의미 도출 (개인 → 우리 모두)
5. 여운 있는 마무리 (이미지/질문/반전)
```

### 변형 패턴
- **A형 (회상)**: 현재 → 과거 회상 → 현재 (깨달음)
- **B형 (관찰)**: 관찰 대상 묘사 → 연상 → 통찰
- **C형 (대비)**: 과거의 나 vs 현재의 나
- **D형 (목록)**: 여러 사례 나열 → 공통 의미 도출
- **E형 (편지)**: 누군가에게 보내는 편지 형식
