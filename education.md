# 교육 콘텐츠 개발

## 커리큘럼 설계 템플릿

### 강좌 기본 구조
```markdown
# [강좌명]

## 개요
- 대상: 초급/중급/고급
- 소요 시간: X시간 (Y주)
- 선수 지식:
- 학습 목표:

## 주차별 커리큘럼

### 1주차: [주제]
- 학습 목표:
- 핵심 개념:
- 실습 과제:
- 퀴즈/테스트:

### 2주차: [주제]
...
```

### 학습 목표 작성 (SMART)
```markdown
## 학습 목표 체크리스트

- [ ] Specific (구체적): 무엇을 배우는가?
- [ ] Measurable (측정 가능): 어떻게 평가하는가?
- [ ] Achievable (달성 가능): 현실적인가?
- [ ] Relevant (관련성): 왜 필요한가?
- [ ] Time-bound (기한): 언제까지?

### 예시
❌ "프로그래밍을 잘하게 된다"
✅ "4주 후 Python으로 간단한 웹 스크래퍼를 만들 수 있다"
```

## 강의 콘텐츠 유형

### 1. 텍스트 강의
```markdown
# Lesson X: [제목]

## 학습 목표
이 강의를 마치면 다음을 할 수 있습니다:
- [ ] 목표 1
- [ ] 목표 2

## 핵심 개념

### 개념 1
[설명]

> 💡 **핵심 포인트**
> 꼭 기억해야 할 내용

### 예제
```code
// 코드 예제
```

## 실습
1. 단계별 실습 가이드
2. ...

## 정리
- 핵심 내용 1
- 핵심 내용 2

## 퀴즈
1. 질문?
   - [ ] A
   - [x] B (정답)
   - [ ] C
```

### 2. 영상 강의 스크립트
```markdown
# 영상 스크립트: [제목]

## 메타 정보
- 예상 길이: X분
- 필요 자료: 슬라이드, 데모 화면

## 인트로 (30초)
"안녕하세요, [인사말]
오늘은 [주제]에 대해 알아보겠습니다.
이 영상을 보시면 [기대효과]"

## 본문

### 파트 1: [소주제] (3분)
[화면: 슬라이드 1]
"[스크립트 내용]"

[화면: 코드 에디터]
"실제로 코드를 작성해보면..."

### 파트 2: [소주제] (3분)
...

## 아웃트로 (30초)
"오늘 배운 내용을 정리하면...
다음 영상에서는 [예고]
좋아요와 구독 부탁드립니다!"
```

### 3. 인터랙티브 튜토리얼
```typescript
// React 기반 인터랙티브 학습 컴포넌트
interface Step {
  id: number;
  title: string;
  content: string;
  code?: string;
  hint?: string;
  solution?: string;
  validator?: (input: string) => boolean;
}

const tutorialSteps: Step[] = [
  {
    id: 1,
    title: "변수 선언하기",
    content: "let 키워드를 사용해 name 변수를 선언하세요.",
    hint: "let 변수명 = 값;",
    solution: 'let name = "홍길동";',
    validator: (input) => input.includes("let name")
  },
  // ...
];
```

## 평가 도구

### 퀴즈 유형
```markdown
## 1. 객관식
다음 중 올바른 것은?
- [ ] A
- [x] B
- [ ] C
- [ ] D

## 2. 빈칸 채우기
JavaScript에서 변수를 선언할 때 사용하는 키워드는 ___, ___, ___ 가 있다.
정답: let, const, var

## 3. 코드 작성
다음 조건을 만족하는 함수를 작성하세요:
- 입력: 숫자 배열
- 출력: 합계
```

### 루브릭 (채점 기준표)
```markdown
| 기준 | 우수 (5점) | 보통 (3점) | 미흡 (1점) |
|------|-----------|-----------|-----------|
| 정확성 | 에러 없음 | 일부 에러 | 실행 불가 |
| 가독성 | 명확한 변수명 | 보통 | 이해 어려움 |
| 효율성 | 최적화됨 | 작동함 | 비효율적 |
```
